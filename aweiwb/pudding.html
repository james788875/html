    基礎指令
    /carry 扛人
    /takehostage 挾持
    /doktor ai醫護

    /carradio 外過音響

    街頭賽車
    創建比賽:
    1.使用指令[/race record] - 然後按p打開地圖標點,雙擊某一個點取消標點
    2.使用指令[/race save name] - name=你想定義的名字.
    3..使用指令[/race clear] - 刪除現有的標點
    啟動比賽:
    1.使用指令[/race list] - 查看所有比賽的名稱
    2.使用指令[/race load name] - 加載你想開始的比賽
    3.使用指令[/race start amount] - amount=參賽金錢,贏得比賽後,得到所有玩家參賽的錢(玩家越多贏的越多)
    玩家操作:
    1.使用指令[/race cancel] - 在開始之前取消已創建的比賽並退款
    2.使用指令[/race leave] - 退出現役比賽（不退款）

    組隊
    創建比賽:
    / crew –此命令將打開一個包含以下所有選項的常規菜單
    / joincrew –您可以使用此命令加入受邀參加的人員
    / createcrew NAME-您可以使用此命令創建一個人員
    / invitecrew –您可以使用此命令邀請某人加入您的人員
    / leftcrew –您可以使用此命令離開
    / provecrew –您可以使用此命令來提升人員
    / demotecrew –您可以使用此命令從人員中刪除升遷
    / listcrew –您可以使用此命令查看人員列表
    / kickcrew –您可以使用此命令將人員踢出
    / deletecrew –您可以使用此命令刪除人員
    / rank –您可以使用此命令查看機員等級

  • 保障玩家權益條規

    保障玩家權益條規
    1.若任何人要綁上班員警.醫護.技工,沒有直接妨礙到他們公職工作跟合理範圍才可進行動作
    2.不可再醫護救起後10分鐘內再次擊殺，判定標準為群體或個人，即公職或個人，視情況判定標準
    3.同一人被買兇，三天不可超過一次，不論買兇者是否為同一人
    4.任何公職門口禁止聚集人群和恣意停車
    5.警局內牢獄不可存在任何非員警綁人.開槍.拳腳攻擊
    6.不可於安全區外射向安全區內
    7.不可勒索.綁架新市民(定義為5天內)
    8.若線上該公職已經呈現缺人狀態,不可勒索.綁架
    9.若線上無醫護.或有醫護卻遲遲未接單(疑似掛機行為,基本判定為5~7分鐘未接單) 被警察追緝 離線重開需自行至警局報到 (假設有醫護,將你救起,警察勢必是直接逮捕,為模擬此情況,特立此規定)
    10.公職人員上班不可打副本(黑幫及酒店暫除外)
    11.嚴禁汙辱其他玩家(問候親戚)違者永ban

  • 搶劫須知

    商店搶劫
    1.不可打野 僅能於商店內部
    2.超商搶劫開始後劫方在公頻公告人數
    3.人數警方可比搶劫方多一人 (建立於沒有人質的情況,若有人質,搶方可選擇少一人)
    4.最多一方3人參戰
    5.地面戰,警方不可出動直升機
    6.劫方酬勞在50萬內
    7.罰單30萬
    8.超商搶案結束後劫方與警方先死的須在公頻發送END
